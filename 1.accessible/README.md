## Vanilla Javascript base Custom Modal

### 개요

React 를 기반으로 재사용이 가능한 modal 을 구현하거나, 라이브러리를 사용해본 적은 있지만, html 내장 api 를 직접적으로 사용해보거나 이 외 프론트엔드적으로 신경써야 하는 부분들에 대해서 검토해보고자 하는 과정들을 기록합니다.
